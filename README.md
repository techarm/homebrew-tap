# techarm/homebrew-tap

Homebrew tap for [techarm](https://github.com/techarm)'s CLI tools.

## Usage

```bash
brew tap techarm/tap
brew install pixa
```

## Available formulas

- **[pixa](https://github.com/techarm/pixa)** — fast Rust image processing CLI

Formulas in this repository are generated and updated automatically by
[cargo-dist](https://github.com/axodotdev/cargo-dist). Do not edit them
by hand.
